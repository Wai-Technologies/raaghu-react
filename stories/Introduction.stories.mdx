### Welcome to Raaghu Design System - React

<div className="containerContent">
 <p><strong>Raaghu</strong> is an open-source React based micro front-end design system. Raaghu employs Bootstrap5 and Storybook 
 features to develop highly customizable UI components and built-in pages. Raaghu provides as a online repository for the developers
 containing the elements, components, map and charts.</p>

<img className="demo" src={demo} alt="flow" />

Through the storybook you can visualize and navigate around the features of
the Raaghu. Snippets for the elements allow you to test and design your UI
faster and efficiently.

</div>

### Why Raaghu?

<div className="containerContent">
  Raaghu Design System bridges the gaps between design and development while
  maintaining a great degree of standardization.
</div>
<ul className="containerContent">
  <li>
    <strong>Seamless UI:</strong> Raaghu contains ready to use templates for the
    ASP net and ABP framework. Fully customizable with the help of our Adobe XD
    and figma design kits.
  </li>
  <li>
    <strong>Charts, Layouts and Grids:</strong>:With more than 60+ react based
    elements, components , pages and charts , Raaghu offers a wide array
    catering to the needs for your UI components.
  </li>
</ul>

### How Raaghu Works?

<div className="containerContent">
  Raaghu leverages micro-front-end capabilities by breaking the page design into
  the components. Based on the wireframe and theme design, Raaghu applies the
  same CSS to elements, components and pages available in the Raaghu.
</div>
<img className="how" src={image} alt="flow" />
<div className="containerContent">
  Thought the Ci/Cd pipelines the components are deployed to the wireframe
  design and form a page
</div>

### Features

<ul className="containerContent">
  <li>
    <strong>Fast: </strong>Achieve up to 3X acceleration with front end
    development through modular creation and component composition.
  </li>
  <li>
    <strong>Consistent : </strong>Get a unified experience via smooth
    integration, code maintenance and consistency in UI.
  </li>
  <li>
    <strong>Scalable : </strong>Provides better scalability as each codebase is
    much simpler to build, test and maintain due to clarity of work allocation.
  </li>
  <li>
    <strong>Maintainable : </strong>Replace and rewrite and update components
    easily even when maintaining large applications. Components allow for easier
    user testing.
  </li>
  <li>
    <strong>Flexible : </strong>Components are inherently independent of the
    tech stack allowing multiple components and features to be deployed rapidly.
  </li>
  <li>
    <strong>Pixel-perfect : </strong> Designed with pixel perfect precision and
    accessibility in mind, with multiple well designed and scalable User
    Interfaces.
  </li>
</ul>
